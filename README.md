# Vypr

Vypr is a Python-syntax-compatible interpreter with **strong type enforcement**.  

The goal of Vypr is to combine the flexibility of Python with the safety of static types:  

- Variables without type annotations behave dynamically, just like in Python.  
- Variables with type annotations are **locked to their type** for their lifetime, preventing unintended reassignments.  
- Compatible with existing Python syntax, making it easy to adopt.
